# BizInsight 360

BizInsight 360 is an AI-powered complaint intelligence platform that helps organizations analyze complaints, identify recurring issues, and gain actionable business insights through an interactive dashboard.

## Features

* Complaint submission and tracking
* Complaint categorization
* Severity assessment
* Business insight generation
* Interactive analytics dashboard
* Category-wise complaint distribution
* CSV-based complaint storage

## Tech Stack

* Python
* Streamlit
* Pandas
* Plotly
* Google Gemini AI

## Project Structure

```text
BizInsight360/
│
├── app.py
├── complaints.csv
├── requirements.txt
├── .gitignore
└── README.md
```

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Use Case

The platform helps businesses understand customer and employee concerns, identify high-frequency problem areas, and make informed decisions based on complaint trends.
